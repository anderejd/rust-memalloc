# memalloc

> Raw allocation APIs in stable rust.

## Usage

See the tests at the bottom of `src/lib.rs` for some examples.

Use the crates.io repository; add this to your `Cargo.toml` along
with the rest of your dependencies:

```toml
[dependencies]
memalloc = "*"
```
